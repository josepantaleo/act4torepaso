🛠️ Pasos previos: Configuración en Visual Studio Code

  1. Crear la carpeta de trabajo: Abrí el explorador de archivos de tu computadora, ingresá a Mis Documentos y creá una carpeta nueva llamada exactamente: repasohtmlcss4to.
  2. Acceder al sitio web: Hacé clic en el siguiente enlace para abrir el proyecto base en una pestaña nueva: Sitio Web para investigar.
  3. Ver el código fuente: Una vez dentro de la página, hacé clic derecho en cualquier parte vacía de la pantalla y seleccioná la opción "Ver código fuente de la página" (o presioná las teclas Ctrl + U). Se abrirá otra pestaña con el código.
  4. Copiar todo el código: Seleccioná absolutamente todo el código visible presionando Ctrl + A en tu teclado y luego copialo con Ctrl + C.
  5. Guardar el archivo en VS Code: Abrí Visual Studio Code, creá un archivo nuevo y pegá el código (Ctrl + V). Al momento de guardarlo (Ctrl + S), buscalo dentro de Mis Documentos, ingresá a la carpeta repasohtmlcss4to que creaste en el paso 1 y guardalo con el nombre actividad.html.
  6. Visualiza en el navegar
  7. Luego Completa el siguiente custionario haz cambios en el codigo si es necesario



1. ¿Para qué sirve la etiqueta <meta charset="UTF-8">?
(Pista: Probá borrarla y fijate qué pasa si escribís palabras con acentos o la letra Ñ en los párrafos).
2. ¿Cuál es la diferencia entre usar un título con <h1> y un título con <h2> según lo que observás en la pantalla y en el código?
3. Si querés agregar un nuevo tema favorito a la lista actual (por ejemplo, "🎬 Edición de Video"), ¿qué etiqueta de HTML tenés que duplicar y escribir dentro de <ul>?
4. Buscá una foto que te guste en Google, hacé clic derecho, seleccioná "Copiar dirección de la imagen" y pegala dentro del atributo src="..." de la etiqueta <img>.
¿Lograste cambiar la foto del proyecto con éxito?
5. ¿Qué sucede si borrás el atributo alt="..." de la etiqueta <img>?
¿Afecta a la visualización de la imagen o cumple otra función?
6. En HTML, las listas <ul> son "desordenadas" (usan viñetas).
¿Qué etiqueta tendrías que usar en lugar de <ul> para que la lista muestre números ordenados?
7. ¿Qué pasaría si te olvidás de cerrar la etiqueta </div> al final del código?
Intentá borrarla y mirá si el diseño se rompe o cambia de lugar.
8. Si quisieras agregar un enlace (link) que te lleve a otra página web,
¿qué etiqueta de HTML deberías investigar?
9. ¿Por qué todo el contenido visible de la página está encerrado dentro de <body> y </body>?
¿Qué pasa si ponés un párrafo fuera de ellas?
10. ¿Cómo se escriben los comentarios en HTML en comparación con CSS?
Explicá qué símbolos se usan.
11. Si quiero cambiar el tipo de letra por Impact o Courier,
¿en qué parte del CSS debo modificar font-family?
12. Buscá margin:20px auto dentro de .contenedor.
Si cambiás 20px por 100px, ¿qué sucede?
13. ¿Qué pasa si escribís margin:20px 0?
¿Qué función cumple auto?
14. Cambiá border:2px solid #333333 por un borde de 8px dashed.
¿Qué aspecto tiene ahora?
15. La propiedad border-radius:8px está en la foto.
¿Qué valor usarías para hacerla completamente redonda?
16. En la lista hay border-left:5px solid #16a085.
¿Qué pasa si cambiás border-left por border-right o border-top?
17. Si cambiás padding:25px por padding:0px,
¿qué pasa con el texto respecto a los bordes?
18. Si cambiás background-color:#e3edf7 por red o yellow,
¿el navegador lo entiende?
19. Modificá line-height:1.6 a 3.0 y luego a 0.8.
¿Qué controla esta propiedad?
20. Si cambiás width:800px por width:100%,
¿la hoja sigue centrada o cambia completamente?
